# Fragmento-Estelar
Fragmento Estelar (FE) es una herramienta de desarrollo de instalar sistemas operativos. Escrito en el nuevo lenguaje de programación Infernal.

Al estar escrito en Infernal, se puede aprender Infernal con este AppImage.

Uso:
sudo ./Fragmento_Estelar-x86_64.AppImage create --os=debian --codename=trixie --path=/mnt/hdd

Esto instalaría la base normal de Debian 13 (Trixie) en /mnt/hdd (lo que parece ser un punto de montaje de un disco duro).
La base normal trae muchas cosas útiles, pero no trae entorno de escritorio.
